# Сброс базы данных

Если в БД остались тестовые данные от предыдущих запусков — удали файл базы:

**Windows:**
```
del backend\data\weave.db
```

**Mac / Linux:**
```
rm backend/data/weave.db
```

После этого перезапусти сервер — база создастся чистой с одним аккаунтом администратора.
